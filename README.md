\# E-commerse Doña Juana



\*\*Sistema de ventas en línea de sneakers\*\*, diseñado para mejorar la experiencia de compra de los clientes y optimizar el proceso de ventas digitales de la tienda "Doña Juana".



---



\## Tecnologías utilizadas



\- \*\*Astro\*\* — Framework moderno para sitios rápidos y eficientes

\- \*\*React\*\* — Interfaz dinámica y componentes reutilizables

\- \*\*Node.js\*\* — Backend y lógica del servidor



---



\## Características principales



\- Catálogo de sneakers

\- Detalle de productos

\- Carrito de compras

\- Posible integración de autenticación (futuro)

\- Integración para pagos en línea (futuro)
